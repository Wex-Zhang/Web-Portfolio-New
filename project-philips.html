飞利浦智慧家 2.0 · 扫拖机器人板块体验设计升级

UX / App Design Philips 实习项目

「品质创新生活家」是飞利浦家居系列的追求。飞利浦智慧家 App 致力打造智能家居控制中枢，整合旗下多款智能家居产品，使用户能在一个平台上无缝管理多个设备。智能家居终端操作系统的体验也直接影响产品销量，因此提供更好的操作体验至关重要。

设计背景

在智能家居生态中，不同产品往往具备各自独特的交互模式，这一碎片化体验会影响用户使用便捷性。本次设计旨在提供更流畅、统一的操控体验，让用户无需在多款 App 间切换，形成连贯的使用习惯。

问题发掘

体验分散

不同智能家居产品对应不同 App，交互逻辑不统一，用户需要重复学习，难以形成连贯体验。

操作低效

功能缺乏清晰分类与层次，核心操作路径复杂，交互繁琐，降低便捷性与效率。

信息弱化

界面采用功能卡片布局，关键信息难以突出，颜色单一、缺乏活力，影响用户对核心功能的关注度。

设计目标

  • 更统一的界面设计：统一不同智能家居设备界面风格，减少割裂感
  • 清晰的信息传递：强化关键信息层次，减少认知负担，提升可读性
  • 流畅的交互与操作路径：精简功能分类，核心功能一目了然
  • 打造独特风格：结合产品特性打造个性化设计，增强品牌识别度

设计产出

用户引入 Onboarding

快速清洁模式、智能管家模式、AI 功能介绍等引导页，帮助用户理解机器人能力；小飞智能引导在地图上绘制禁拖区，优化首次使用体验。

功能导览 Function Guide

引导用户熟悉更多设置、视频模式、小飞智能、快速建图等核心模块，确保新用户能快速上手。

快速建图 Map Building

建图指南、地图编辑（房间合并、分割、禁区、地面材质、重命名）、建图进度展示，支持一楼/多楼层管理。

清洁偏好 Clean Preferences

全局设定：模式（扫拖一体 / 先扫后拖 / 扫地 / 拖地）、吸力、水量、清扫次数；房间设定：按房间自定义参数，支持拖动调整清扫顺序。

基站看板 Station Kanban

机器人在基站时显示基站看板：一键自清洁、烘干拖布、清水箱 / 尘袋 / 污水箱状态；机器人工作时显示「返回充电」模式切换。

视频巡航 Video Monitoring

选点直达、视频巡航、语音通话、返回基站；巡航任务设置（定时全屋巡航、异响触发）；云端相册、局部清洁等辅助功能。
